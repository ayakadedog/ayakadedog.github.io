633. Sum of Square Numbers

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。

输入输出样例
输入是两个字符串 S 和 T，输出是一个 S 字符串的子串。

解析
题目要求我们找该数的平方和，那就很简单了啊，直接双指针，一个从头开始一个从尾开始，尾部那个考虑到从原数开始的话再平方一下直接就远超了，那我们不妨直接把sqrt(c)作为尾，然后一前一后向中间运动，当前>后时还没有合适的值出现时，那就说明没有能构成该数的两数平方和。
